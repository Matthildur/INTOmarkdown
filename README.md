## 2. Uppsetning á vim && git

Sigmar:
Það var frekar einfalt að setja þetta upp. Reyndar þurfti ég að byrja á að setja upp sudo:

su
apt-get install sudo
CTRL + D

Þá gat ég gert sudo apt-get install vim og sudo apt-get install git.
